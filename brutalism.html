Mood

Raw, unapologetic, anti-smooth — intentionally ugly by conventional standards. Honesty over beauty.

Era

1950s–1970s (architectural brutalism). Web brutalism: 2010s–present, a reaction against overly polished web design.

Colors

High-contrast, clashing — bright yellow (#ff0), cyan (#0ff), magenta, black. No subtlety, no gradients, no mercy.

Typography

Default system fonts (Arial/Helvetica). No web fonts, no elegance. Typography serves function, not aesthetics.

Characteristics

Raw HTML defaults, thick borders, heavy box-shadows, clashing colors, no subtle transitions, intentionally crude layout.
